## 🎯 API ENDPOINTS

```
┌─────────────────────────────────────────────────────────────────┐
│ ENDPOINT 1: GET /simulate                                       │
├─────────────────────────────────────────────────────────────────┤
│ Purpose: Get 24-hour supply/demand/battery profile             │
│ Called By: getGridStatus() in mobile app                       │
│ Response: { total_supply_kwh, demand_kwh, battery_pct }        │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ ENDPOINT 2: POST /grid-status                                   │
├─────────────────────────────────────────────────────────────────┤
│ Purpose: Get grid health status + recommended action           │
│ Called By: getGridStatus() in mobile app                       │
│ Request:  { battery_level_pct, current_supply_kwh,            │
│             current_demand_kwh }                               │
│ Response: { status, recommended_action }                       │
│ Status:   HEALTHY | WARNING | CRITICAL                         │
│ Action:   STABLE | CHARGE_BATTERY | STORE | LOAD_SHED         │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ ENDPOINT 3: GET /forecast                                       │
├─────────────────────────────────────────────────────────────────┤
│ Purpose: Get 12-24 hour supply/demand forecast                 │
│ Called By: getForecast() in mobile app                         │
│ Response: { horizonHours, points[] }                           │
│ Points:   { timestamp, supply, demand }                        │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ ENDPOINT 4: POST /predict                                       │
├─────────────────────────────────────────────────────────────────┤
│ Purpose: Get ML-recommended actions from arrays                │
│ Called By: getPrediction() in mobile app                       │
│ Request:  { solar_output_kwh[], wind_output_kwh[],            │
│             demand_kwh[], battery_level_pct,                  │
│             forecast_horizon_hours }                           │
│ Response: { recommended[], source }                            │
│ Source:   recommender | rule_based                             │
└─────────────────────────────────────────────────────────────────┘
```
